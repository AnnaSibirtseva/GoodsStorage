	Login: admin 
	Password: 12345
2. Чтобы сменить пользователя нажмите на его ник сверху, слева от покупок
     (выйти из аккаунта продавца можно через меню и кнопочку 'Log out')
3. Чтобы оплатить заказ нажмите левой кнопкой мыши на любую ячейку связанную с эти заказом (номер, статус, цена)
     (аналогично работает для продавца с изменением статуса заказа)
4. Когда вы заходите, писать домен электронной почты не надо.
     (например: admin@mail.ru -> admin)
